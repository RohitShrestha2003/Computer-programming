### Problem Description
Beckett Pizza Plaza offers a **4-for-3** deal where customers ordering four pizzas receive the **cheapest pizza for free**.

The program:
- Prompts the user to enter the prices of **four pizzas**
- Ensures all entered prices are **valid positive numbers**
- Calculates the **final amount to pay**
- Displays the **percentage discount achieved**

### Key Features
- Handles invalid input (letters, zero, negative values)
- Automatically identifies the cheapest pizza
- Correctly applies the discount
- Outputs results formatted as currency

### Problem Description
This program simulates a simple password verification system.

The program:
- Prompts the user to enter a password
- Rejects passwords shorter than **9 characters**
- Asks the user to correctly identify **three randomly selected characters** from the password
- Terminates immediately if any check fails

### Key Features
- Minimum length enforcement
- Randomised character position checks
- Immediate failure on incorrect input
- Clear success and failure messages
